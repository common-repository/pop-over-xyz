== Description ==
Introducing PopOverXYZ, a simple light weight WordPress plugin that allows you to show beautiful tool tips over any text by using simple short tags. Simply install it, activate it and start using short tags to generate text pop overs.


A detailed list of all functions can be found on <a href="https://www.popover.xyz" target="_blank">https://www.popover.xyz</a>.

**SUPPORT**
The plugin is constantly being developed and supported. Support inquiries please only by email <a href="mailto:office@webmuehle.at">office@webmuehle.at</a>


== Installation ==

Upload the plugin via ZIP or install and activate it via the WordPress plugin installer. Afterwards all settings can be adjusted.
